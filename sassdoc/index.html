    General

    variables

    breakpoint-medium

    $breakpoint-medium: 1024px !default;

    Description

    Media Breakpoints mobile friendly hybrid display for mobile friendly, hybrid display will show medium on touch devices for screen widths up to hybrid-medium and screen heights less than hybrid-medium-height hybrid display will show large on desktop screens from medium while orientation:landscape and screen height greater than hybrid-medium-height

    Breakpoint

    $Breakpoint: (
        medium:$breakpoint-medium,
        medium-portrait:$breakpoint-medium-portrait,
        large:$breakpoint-large,
        small:$breakpoint-small,
        small-portrait:$breakpoint-small-portrait,
        tiny:$breakpoint-tiny,
        x-large:$breakpoint-x-large,
        tabs:$breakpoint-tabs,
        enforce-hybrid-display:$enforce-hybrid-display,
        hybrid-medium:$breakpoint-hybrid-medium,
        hybrid-medium-height:$breakpoint-hybrid-medium-height
);
